# tech-track-23-24
Repository voor de Tech track voor minor Information Design 23-24.

Om deze applicatie te runnen moet je de folder my-app downloaden. Open dan in terminal deze directory en run de command 'npm i', dan zullen alle benodigde modules zich installeren, en als laatste 'npm run dev' om de website te kunnen bekijken op een localhost.
Anders kun je de link bekijken in de sidebar, waar mijn website live staat :)
